# Miroslav Vanco Studio
## Vizia e-shopu na predaj knih

Verzia dokumentu: 1.0

---

# Ciel projektu

Vytvorit jednoduchy, spolahlivy a lahko rozsiritelny internetovy obchod pre vydavatelstvo Miroslav Vanco Studio.

System musi umoznovat predaj:

- tlacenych knih
- elektronickych knih
- audioknih

Projekt musi byt navrhnuty tak, aby bolo mozne bez programovania pridavat dalsie knihy a dalsie jazykove verzie.

Majitel projektu nie je programator. Administracia musi byt preto co najjednoduchsia.

---

# Filozofia projektu

Projekt ma byt:

- jednoduchy
- rychly
- prehladny
- bez zbytocnych kniznic
- lahko pochopitelny

Prednost ma citatelny kod pred komplikovanymi rieseniami.

Kazda dolezita funkcia ma byt komentovana po slovensky.
